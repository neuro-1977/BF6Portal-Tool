## FORMATTING GUIDE

### Block Types:
- **ACTION**: Rectangular block that performs an action (e.g., "Send Message", "Teleport Player")
- **VALUE**: Rounded block that returns a value (e.g., "Get Player Health", "Random Number")
- **C_SHAPED**: C-shaped wrapper block for control flow (e.g., "If", "While", "For")
- **CONDITION**: Boolean condition block (e.g., "Player Is Alive", "Team Equals")

### Parameter Icons:
- üë§ = Player
- üéØ = Target
- üìç = Location/Position
- 123 = Number
- ABC = Text/String
- ‚öô = Boolean/Condition
- üîß = Dropdown/Option
- ‚è± = Time/Duration
- üé® = Color
- üì¶ = Array
- üîä = Audio

### Example Entry:
```
- [x] Block Name: Set Player Health
  - Type: ACTION
  - Parameters:
    - Player: üë§ (dropdown/player selector)
    - Health: 123 (number input, 0-100)
  - Description: Sets the health of the specified player to the given value
  - Notes: Health is clamped between 0-100
```
