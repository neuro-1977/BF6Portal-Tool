# Command Pallet: Quick Reference

Use these @commands in your prompts to target specific environments, workflows, or databases. Just type @command in your message.

## Core Environments
@knowledgebase   - Reference the knowledgebase (all extracted info, screenshots, assets, help, etc.)
@dump            - Reference the Dump folder (incoming screenshots/assets to be processed)
@blockly         - Reference the Blockly editor, blocks, or blockly-specific UI
@webui           - Reference the web UI (HTML/CSS/JS, splash, desktop, etc.)
@concepts        - Reference the concepts database (ideas, mockups, design notes)
@assets          - Reference the assets folder (images, sounds, sprites, etc.)
@docs            - Reference documentation (README, master prompt, etc.)
@old             - Reference the old folder (legacy files, to be scrubbed/archived)

## Workflow Shortcuts

## Special
@todo            - Reference or update the current todo list
@prompt          - Reference or update the master prompt
@review          - Request a review of recent changes or workflow

## YOLO & Pallet
@yolo            - Toggle YOLO mode on/off (short, rapid, key-point updates)
@pallet          - Add or manage items in the command pallet (quick-access commands, workflow shortcuts)
