# Screenshot Analysis: Missing Info & Requests

This file lists what information is still needed or missing from the current set of screenshots and extracted help tips. Please review and add any details you can, or upload new screenshots as needed.

## What I Need / What's Missing

- Screenshots of blocks in all possible snap/connection states (e.g. input/output, nested, chained, etc.)
- Widget options for each block type (dropdowns, color pickers, toggles, etc.)
- Full help tips for every block (some blocks have partial or missing help text)
- Examples of complex block arrangements (multi-block logic, event chains, etc.)
- Screenshots showing error states or invalid block connections
- Screenshots of rarely used or advanced blocks
- Any official documentation or reference images for custom blocks
- Screenshots with clear block names and categories visible
- Screenshots of the UI with tooltips or context menus open
- Any missing block types not yet covered in the current screenshot set

## How to Help
- Upload new screenshots to the Dump folder as you find them
- Add notes or filenames to this file if you have extra context
- If you know what a screenshot is for, add a comment here or move it from "unsorted" to the correct category

This file will be updated as new screenshots are processed and more info is extracted. Thank you!
